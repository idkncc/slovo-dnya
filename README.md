# Слово дня
~~глупы~~ познавательный канал в телеграме

## ноте
Срабатывает каждый день в ~8:00 по МСК, благодаря `on: schedule` в [`.github/workflows/publish-word.yaml`](.github/workflows/publish-word.yaml).
